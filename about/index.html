    The blog

    The main topic of this blog is architecture in the context of software engineering. The blog has a technical approach, therefore topics like software construction, infrastructure, Linux, and programming paradigms are covered (among others).

    Everything that relates somehow to software architecture, is a topic suitable for this blog.

    The author

    Mariano is a software engineer who spends most of his time creating software with Python, and mentoring fellow programmers. Mariano's main areas of interests besides Python are Linux, software architecture, functional programming, distributed systems, and speaking at conferences.

    He is the author of the book Clean code in Python
